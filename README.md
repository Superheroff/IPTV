# 运行
- 安装nodejs
- 安装crypto-js
    ```noedjs
    npm install crypto-js
    ```
- 安装python
- 安装requirements.txt
    ```
    pip install -r requirements.txt
    ```
- 运行
    ```run
    python madou.py
    ```
